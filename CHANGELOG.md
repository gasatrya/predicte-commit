# Change Log

All notable changes to the "predicte-commit" extension will be documented in this file.

## [0.0.1]

### Added

- Initial release of Predicte Commit.
- Generate commit messages from staged diffs via SCM button or command palette.
- Support for Mistral AI (cloud) and Local LLMs (Ollama-compatible).
- Configuration for ignored files (`predicteCommit.ignoredFiles`).
- Automatic diff truncation and payload sizing.
- Secure API key storage integration.
- Output channel for debug logging.
- Robust handling of model output (stripping fences, normalizing headers).
